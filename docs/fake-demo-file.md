# Fake Demo File

This is a demo file added by Devin to demonstrate screen recording on a Windows VM.

## Purpose

This file was created as part of a demo to show that:

1. **Screen recording works on Windows VMs** - not just Linux
2. **Computer use** can interact with the OtterWorks UI
3. **Video generation** via ffmpeg works on Windows Server 2022
